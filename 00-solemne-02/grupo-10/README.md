## Descripción textual del proyecto

Paso a paso en clase

El objetivo es tener un código de enviar desde un Raspberry Pi 2W y un código de recibir en un Arduino Uno R4 Wifi, utilizaremos un botón como una especie de "Gate" para tener la opción de activar y desactivar el envío de lecturas hacia Adafruit IO, de esta manera, el servidor de IO no colapsa y evitamos problemas.

Conexión de botón a Raspberry Pi Pico 2w

Lo primero que debemos hacer es lograr prender un led mediante un potenciómetro, después de eso podemos avanzar hacia mover un servo, de momento solo led.

De momento pudimos prender el led en el arduino mediante el pulso con un botón en raspberry pi
