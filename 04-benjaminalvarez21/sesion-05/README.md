### Hardware

Corresponde a todo lo físico y tangible que se puede tocar, como el CPU, el monitor o la placa Arduino.

### Software

Es intangible y lógico. Incluye programas y aplicaciones como Windows o plataformas como Microsoft.

## Instalación de Adafruit IO en Arduino

Para trabajar con Adafruit IO es necesario instalar sus librerías desde el Library Manager del Arduino IDE.

Es importante instalar todas las dependencias que solicita ya que son necesarias para que el sistema funcione correctamente.

## Pasos de configuración

- Crear un archivo nuevo (New File) en Arduino IDE

- Copiar el código proporcionado en Discord

- Crear una nueva pestaña (New Tab) llamada `config.h`

- Ir al repositorio en GitHub `solemne 1 / grupo 11 / enviar / config.h`

 - Copiar el contenido de `config.h` y pegarlo en la nueva pestaña dentro de Arduino

Esto permite configurar correctamente la conexión con Adafruit IO y preparar el envío de datos.
