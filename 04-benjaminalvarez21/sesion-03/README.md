## Enfoque del curso

El curso se centra en la comunicación entre dispositivos, trabajando con Arduino Uno R4 WiFi como emisor y Raspberry Pi Pico 2 W como receptor.

El objetivo es que el sistema funcione completo, es decir, que logre enviar y recibir datos correctamente.

Se trabaja en Visual Studio Code utilizando C++, Python y MicroPython.

## Microcontroladores vs Computadores

#### Microcontroladores (Arduino)

- Simples
 
- Tareas específicas

- Menor potencia

#### Computadores (Raspberry Pi)

- Más complejos

- Más rápidos

- Procesos más avanzados

#### Lógica de programación

- Arduino -> setup() y loop()

- Raspberry Pi Pico -> main() + while(true)

## Raspberry Pi

La Raspberry Pi Foundation (2009) busca crear computadores accesibles.

#### Raspberry Pi Pico 2 W

- Chip RP2350

- Doble núcleo

- Wi-Fi + Bluetooth

- Bajo costo y buen rendimiento
